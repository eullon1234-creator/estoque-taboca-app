# Controle de Estoque - UHE Taboca

Aplicacao web para controle de materiais de almoxarifado, com foco em operacao rapida, rastreabilidade e reducao de faltas de estoque.

## Repositorio

- GitHub: [estoque-taboca-app](https://github.com/eullon1234-creator/estoque-taboca-app)

## Acesso ao app

- Link de producao: [Abrir aplicacao](https://eullon1234-creator.github.io/estoque-taboca-app/)

## Para que serve

Este sistema foi criado para facilitar a rotina de estoque da equipe, permitindo:

- cadastro e organizacao de produtos;
- controle de entradas e saidas;
- historico de movimentacoes para auditoria;
- controle de requisicoes de material;
- acompanhamento de itens com estoque baixo;
- apoio em relatorios e tomada de decisao.

## Principais funcionalidades

- Painel com indicadores de estoque
- Cadastro, edicao e exclusao de itens
- Registro de entrada com nota fiscal e fornecedor
- Registro de saida com responsavel e local de aplicacao
- Gestao de requisicoes
- Log de saidas
- Controle de baixa RM
- Cadastro de locais
- Exportacoes e backup

## Tecnologias

- HTML
- Tailwind CSS
- JavaScript (ES Modules)
- Firebase Firestore

## Como executar localmente

1. Clone o repositorio:

```bash
git clone https://github.com/eullon1234-creator/estoque-taboca-app.git
```

2. Abra a pasta no VS Code.
3. Use uma extensao de servidor local (ex.: Live Server) para abrir o projeto.

## Publicacao no GitHub Pages

1. Acesse Settings > Pages no repositorio.
2. Em Build and deployment, selecione Deploy from a branch.
3. Selecione a branch principal e pasta /(root).
4. Salve e aguarde o link publico ser gerado.
